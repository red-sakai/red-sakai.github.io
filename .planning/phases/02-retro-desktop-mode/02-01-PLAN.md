  <name>Task 1: Extend useWindowManager with browser type and Record refactor</name>
  <files>app/desktop/hooks/useWindowManager.ts</files>
  <action>
    Per D-05 (IE-style browser) and D-06 (iframe), the window manager's WindowState.component union must accept `"browser"`.

    Two changes to `useWindowManager.ts`:

    1. **Type union extension (line 9):** Add `"browser"` to the `component` union type — change from `"about" | "portfolio" | "explorer" | "paint" | "controlpanel"` to `"about" | "portfolio" | "explorer" | "paint" | "controlpanel" | "browser"`.

    2. **Refactor openWindow() title/icon resolution (lines 27-50):** Replace the if-else chain for `title` and `icon` with Record lookup maps. This is more maintainable than 6 ternary branches:

       ```typescript
       const offsets: Record<string, number> = {
         about: 0, portfolio: 15, explorer: 30, paint: 60, controlpanel: 90, browser: 45
       };
       ```

       Add title and icon Record maps:
       ```typescript
       const titleMap: Record<string, string> = {
         about: "About Me - Notepad",
         portfolio: "My Portfolio",
         explorer: "File Explorer",
         paint: "Paint",
         controlpanel: "Control Panel",
         browser: "Internet Explorer",
       };
       const iconMap: Record<string, string> = {
         about: "📄",
         portfolio: "🏠",
         explorer: "📁",
         paint: "🎨",
         controlpanel: "⚙️",
         browser: "🌐",
       };
       ```

       Then the newWin assignment becomes:
       ```typescript
       const newWin: WindowState = {
         id,
         title: titleMap[component] || component,
         icon: iconMap[component] || "📄",
         component,
         width: component === "portfolio" ? Math.round(window.innerWidth * 0.8) : 500,
         height: component === "portfolio" ? Math.round(window.innerHeight * 0.8) : 380,
         x: component === "portfolio" ? Math.round((window.innerWidth - Math.round(window.innerWidth * 0.8)) / 2) : 50 + offset,
         y: component === "portfolio" ? Math.round((window.innerHeight - Math.round(window.innerHeight * 0.8)) / 2) + 15 : 30 + offset,
         zIndex: zCounter.current++,
         minimized: false,
         maximized: false,
       };
       ```

    **What to avoid:** Do not change the function signature or any existing behavior. Do not remove the portfolio special-casing for width/height/position. The Record maps replace only the title/icon ternary chains, not the offsets Record which is already in the correct pattern.
  </action>

  </verify>
